Privacy Policy

SnapDog does not collect, store, transmit, or share any personal data on its own. No analytics, no telemetry, no accounts. Here is the full policy.

Effective date: May 5, 2026

Summary

SnapDog does not collect, store, transmit, or share any personal data on its own. Captures stay on your Mac unless you explicitly configure an upload destination. There are no analytics, no telemetry, no accounts, and no third-party SDKs that phone home.

Data Collection

SnapDog itself collects no data whatsoever. Specifically:

  • No personal information (name, email, phone number)
  • No usage analytics or telemetry
  • No crash reports sent to external servers
  • No advertising identifiers or tracking
  • No cookies or web beacons
  • No location data

On-Device Storage

Captures, recordings, edits, history entries, tags, and favorites are saved locally on your Mac. Files go to ~/Pictures/SnapDog by default (or wherever you configure under Settings → Storage). The history database is a SwiftData store inside SnapDog’s app sandbox container. None of this leaves your Mac.

Upload Destinations (Opt-In)

SnapDog can upload captures to external services only when you configure a destination and either turn on auto-upload or trigger a manual upload. Supported destinations include Imgur, Dropbox, SFTP servers you control, Catbox.moe, 0x0.st, and any ShareX-compatible .sxcu uploader you import. Each upload goes directly from your Mac to the destination’s servers — SnapDog does not proxy, log, or retain copies on any server we operate. The destination’s own privacy policy governs what happens to your data once it arrives there.

API Keys, OAuth Tokens, and Passwords

API keys for upload and optional AI providers, OAuth access + refresh tokens (Dropbox, Imgur), SFTP passwords, SFTP private keys + passphrases, and webhook signing secrets are stored in the macOS Keychain, namespaced to the SnapDog bundle. You can clear them at any time from Settings → Destinations / AI / Webhooks, or remove them directly from Keychain Access.

AI Image Analysis (Optional)

The Built-In Demo processes Describe, Extract Text, Suggest Filename, and transcript summary requests on your Mac with Apple Vision and deterministic local analysis. It requires no account, key, model download, or network. If you explicitly select another cloud or locally hosted provider, SnapDog sends the downscaled capture or processed text directly to that provider using your credential. SnapDog operates no intermediary AI server. The privacy policy of the selected provider governs cloud requests; the Built-In Demo and locally hosted providers keep processing on-device.

Webhooks (Opt-In)

If you configure a webhook URL, SnapDog POSTs JSON metadata about save and upload events to that URL. The payload includes the file name, local path, optional remote URL (if uploaded), file size, extension, and timestamp. No image bytes are sent. If you set a signing secret, the request is HMAC-SHA256 signed via the X-SnapDog-Signature header so your endpoint can verify authenticity.

Microphone, Camera, and Screen Recording

SnapDog uses these macOS permissions only to fulfill the capture or recording you initiate. Audio and video stay on your Mac and are written only to the file you save. The camera is only opened when you toggle it on; the microphone is only opened when you opt in to mic recording during a screen capture.

Children’s Privacy

SnapDog does not knowingly collect information from children under 13. The app contains no user accounts, no social features, and no data collection of any kind.

Third-Party Services

SnapDog uses one third-party Swift package: Citadel (Apache 2.0), which provides SSH/SFTP client code via Apple’s swift-nio-ssh. Citadel makes no network calls except to the SFTP server you configure. There are no analytics SDKs, advertising networks, or external APIs invoked at app launch.

Changes to This Policy

If we ever change this policy, we will update this page and the effective date above. Given that SnapDog collects no data of its own, we do not anticipate material changes.
