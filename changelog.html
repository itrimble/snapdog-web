Latest · Version 1.0.0

The Mac App Store debut

Launching soon on the Mac App Store · macOS 14 (Sonoma) or later

SnapDog’s first Mac App Store release builds on 0.2.0 with a full post-capture automation model, a visual workflow builder, multi-provider AI (including fully local), a wider destination set, and a long list of capture and reliability polish.

Post-capture automation

  • Actions — a reorderable, individually toggleable list of post-capture automations (upload, webhook, Slack, AI), with a master After Capture mode: Save Locally, Run Actions, or Ask Every Time with a floating per-capture chooser.
  • Visual workflow builder — chain steps like Screenshot → OCR → AI Summarize → Upload → Slack as cards. OCR text, AI summaries, and upload links flow between steps automatically, and workflows run detached so they never block a capture.
  • Slack posts the real image — uploads the actual screenshot via a Slack bot token, with an incoming-webhook fallback.

AI, your way

  • Built-In Demo (On-Device) — review every AI path without an account, API key, model download, or network.
  • Saved provider profiles — optional cloud and locally hosted providers can be configured and switched per task.
  • Reasoning models handled cleanly — chain-of-thought tags from models like MiniMax-M3 and DeepSeek-R1 are stripped automatically, so you get just the answer.
  • Visible AI failures — errors open in the result window instead of vanishing in a banner.

Capture

  • System-shortcut takeover — optionally bind SnapDog to the macOS ⇧⌘3 / ⇧⌘4 / ⇧⌘5 screenshot shortcuts, with conflict warnings.
  • Native shutter sound — the familiar macOS click the moment a capture lands (on by default, toggleable).
  • Region capture on every display — the picker now overlays all connected monitors, not just the one under your cursor.
  • Window picker grabs dialogs — modal alerts and floating panels are now selectable, not just normal app windows.

Editor & History

  • Shapes gallery — rounded rectangle, triangle, diamond, pentagon, hexagon, and star, each with Outline, Solid, or Translucent fill styles.
  • Quick Look — preview any capture Finder-style with Space or a double-click, and jump straight into the editor from any History row.
  • Watermark customization — any system font, custom color, size, opacity, and free placement anywhere on the image.

Destinations & reliability

  • Destination set expanded — Amazon S3 and S3-compatibles (R2, B2, MinIO, Spaces), Google Drive, OneDrive, Slack, Discord, Telegram, Notion, Mastodon, Bluesky, Cloudinary, GitHub, iCloud Drive, and AirDrop, each with its own configuration UI.
  • Reliability sweep — Dropbox tokens auto-refresh, Slack returns real public links, Mastodon waits for media processing, GitHub re-uploads no longer fail, and Telegram links no longer expose the bot token.
  • Sturdier foundations — Capture History moved to a dedicated private store, a single-instance guard prevents duplicate copies racing each other, and hotkeys that fail to register now tell you instead of silently doing nothing.
  • Settings reorganized — into Setup, Automation, and System groups, with capture options in their own tab.
Version 0.2.0

Destinations, workflows, AI

The big ShareX-parity release: uploads, OAuth, SFTP, webhooks, beautify, and recording upgrades.

Recording

  • GIF as a first-class recording format alongside MP4 / MOV.
  • Webcam picture-in-picture with a draggable, resizable live preview — plus webcam-only recording and a FaceTime-style mirror toggle.
  • Pause / resume (⌘⇧P) with no gap in the saved file, 3-2-1 countdown, multi-monitor display picker, click-ripple visualizer, and quality presets.

Uploads & sharing

  • Imgur, Dropbox, and SFTP (pure Swift, password or Ed25519 keys, trust-on-first-use host-key pinning), plus anonymous Catbox.moe and 0x0.st.
  • ShareX .sxcu custom uploaders with full format support, and URL shorteners.
  • Per-destination Test, delete-from-remote, a Recent Uploads menu, Copy as Markdown / HTML / BBCode, and Export HTML for your history.

Everything else

  • AI Analyze (bring your own key) — Describe, Extract Text, and Suggest Filename on any capture in History.
  • Beautify — one-click gradient frame, rounded corners, and soft shadow.
  • Pin to Screen floating thumbnails with drag-out, webhooks with HMAC-SHA256 signing and auto-retry, hotkey-triggered workflows, and a multi-step onboarding.
  • Security first — secrets live in the Keychain, OAuth runs through ASWebAuthenticationSession with PKCE, and the App Sandbox stays on.
Version 0.1.0

The foundation

The on-device MVP everything else is built on.
  • Fullscreen, window, and region capture with global hotkeys.
  • Annotation editor, capture history, and watch folders.
  • Screen recording, OCR, QR generate / decode, color picker, and on-screen ruler.
  • Image effects, customizable file naming, and first-run onboarding.

SnapDog is launching soon on the Mac App Store — $14.99 one-time, with $9.99 intro pricing for the first 7 days from launch. Family Sharing on, no subscriptions, no in-app purchases. Requires macOS 14 (Sonoma) or later.
